<DocSection title="Run Manifest and Coverage Checks">
	<CommandBlock
		commands={[
			"python3 scripts/check_language_feature_manifest.py --pyrs target/debug/pyrs --cpython-bin /Library/Frameworks/Python.framework/Versions/3.14/bin/python3 --manifest docs/LANGUAGE_FEATURE_MANIFEST.json --out perf/language_feature_manifest_latest.json",
			"python3 scripts/check_language_feature_coverage.py --inventory docs/LANGUAGE_FEATURE_INVENTORY.json --probe-results perf/language_feature_manifest_latest.json --probe-map docs/LANGUAGE_FEATURE_PROBE_MAP.json --out perf/language_feature_coverage_latest.json --enforce --min-coverage-percent 100 --max-probe-fanout 90",
		]}
	/>
	<Callout title="Interpretation">
		A green manifest/inventory gate means tracked feature probes currently match CPython for covered rows;
		it is not a blanket claim that every runtime edge case outside those probes is closed.
	</Callout>
</DocSection>
