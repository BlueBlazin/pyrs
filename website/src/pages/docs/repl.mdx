<DocSection title="Exit and Interrupt">
	<ul class="m-0 list-disc space-y-1.5 pl-5">
		<li><code>Ctrl+D</code> exits the REPL on macOS/Linux terminals.</li>
		<li><code>Ctrl+C</code> interrupts the running statement where supported.</li>
		<li>Fatal runtime exceptions print traceback output and return to prompt when possible.</li>
	</ul>
	<Callout title="Tip">
		If you need deterministic non-interactive behavior for scripts and CI, prefer `pyrs -c ...` or
		file execution over a REPL session.
	</Callout>
</DocSection>
