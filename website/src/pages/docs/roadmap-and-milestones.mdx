	source, tests, and checked-in probe artifacts.
</DocSection>

<DocSection title="Current Workstreams">
	<DataTable headers={["Workstream", "Current focus"]}>
		<tr>
			<td>Release readiness</td>
			<td>Installation polish, public docs accuracy, supported-platform packaging, and remaining pre-release hardening.</td>
		</tr>
		<tr>
			<td>Runtime + stdlib long tail</td>
			<td>CPython 3.14 parity closure across remaining stdlib/runtime edge behavior and mapped test gaps.</td>
		</tr>
		<tr>
			<td>Native extensions + scientific stack</td>
			<td>C-API lifetime-model closure, NumPy bring-up blockers, and broader extension compatibility for real workloads.</td>
		</tr>
		<tr>
			<td>Performance + architecture</td>
			<td>Observability, benchmark discipline, and deeper VM/runtime cleanup without regressing semantics.</td>
		</tr>
	</DataTable>
</DocSection>
